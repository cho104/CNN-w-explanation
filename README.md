# CNN-w-explanation (C++ Port)

3D visualization of a Convolutional Neural Network (CNN) inference process in C++ w/ hover effect and more intuitive control.

https://github.com/user-attachments/assets/6ec58f69-4ee1-4366-92a3-e48bae92bb7a

## Overview
This project is a C++ port of the original Processing-based [CNN-visualization](https://github.com/okdalto/CNN-visualization) by okdalto. It visualizes the step-by-step feature extraction and classification of a CNN, including:
*   Convolutional Layers
*   Flattening operations
*   Multi-Layer Perceptron (MLP) dense layers

## Build Instructions
```bash
mkdir build
cd build
cmake ..
make
```
